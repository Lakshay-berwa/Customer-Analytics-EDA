# Customer-Analytics-EDA
A complete Data Science project demonstrating Data Acquisition, Data Cleaning, Feature Engineering, Data Profiling, and Exploratory Data Analysis (EDA) using Python, Pandas, NumPy, Matplotlib, Seaborn, SQL, JSON, and API integration.
# Customer Data Preprocessing & EDA Project

## Project Overview

This project demonstrates the complete Data Science workflow, including:

- Data Acquisition
- Data Cleaning
- Data Preprocessing
- Feature Engineering
- Data Profiling
- Exploratory Data Analysis (EDA)

The project uses customer purchase behavior data collected from multiple sources such as CSV files, JSON files, SQL databases, and APIs.

---

## Objectives

The main objectives of this project are:

- Understand different types of tensors using NumPy
- Collect data from multiple sources
- Clean and preprocess raw datasets
- Handle missing values and duplicates
- Perform exploratory data analysis
- Generate meaningful visualizations
- Analyze customer purchasing behavior

---

## Technologies Used

- Python
- Pandas

- Matplotlib
- Seaborn
- SQLite
- Requests Library

## Project Workflow

### Part A – Tensor Basics

Implemented different tensor structures using NumPy:

- Scalar Tensor
- Vector Tensor
- Matrix Tensor
- 3D Tensor

### Part B – Data Acquisition

Data collected from:

- CSV File
- JSON File
- SQLite Database
- REST API

### Part C – Data Cleaning

Performed:

- Missing Value Treatment
- Duplicate Removal
- Data Type Conversion
- Unnecessary Column Removal

### Part D – Exploratory Data Analysis

Visualizations created:

- Age Distribution
- Income Distribution
- Purchase Distribution
- Gender vs Purchases
- Income vs Churn
- Correlation Heatmap
- Pairplot Analysis

### Part E – Data Profiling

Generated:

- Correlation Analysis
- Data Quality Checks
- Duplicate Detection
- Missing Value Reports

---

## Key Insights

- Customer income and purchase behavior were analyzed.
- Correlation between numerical variables was identified.
- Data quality issues such as missing values and duplicates were resolved.
- Visualizations helped understand customer trends and patterns.
